04 / The Tech

Meet Claude Code

Your terminal-based software engineering copilot.

1

Writes raw code files (HTML, CSS, JS) directly on your laptop filesystem.

2

Hosts and boots preview environments so you can see changes live in real-time.

3

Monitors build errors, self-corrects code, and pushes updates using natural language commands.

terminal -- claude-code

💡 Pro-Tip Difference:

claude.ai chat suggests code snippets and answers questions.

Claude Code takes command line actions—editing files, resolving errors, and launching live servers inside your local folder.

07 / Setup Part 1

Install Claude Code

Open your terminal application and execute the appropriate command.

🍎

macOS

  1. Open Terminal (Cmd + Space and search "Terminal").
  2. Paste the installer block below and hit Enter.
  3. Exit and restart Terminal once done.
curl -fsSL https://claude.ai/install.sh | bash
🪟

Windows

  1. Launch PowerShell (search "PowerShell" in Start menu).
  2. Paste the installer snippet below and hit Enter.
  3. Exit and restart PowerShell once done.
irm https://claude.ai/install.ps1 | iex

09 / Crafting the Ask

Anatomy of a Great Prompt

Every high-vibe custom application relies on five critical parameters:

🎯

1. Goal

What students will physically do inside your application.

👥

2. Audience

Specify grades, subjects, or cognitive profiles.

📖

3. Content

Supply raw terms, vocab, math problems, or text data directly.

🛑

4. Limits

Keep it to 1 self-contained HTML file. No login setups.

🎨

5. Aesthetic

Set layout, theme, dark-mode, and playfulness levels.

10 / The Blueprints

The Golden Prompt Blueprint

Copy this exact outline, plug in your subject data, and send it to Claude Code.

Build me a single-page web app called [APP NAME] for my [GRADE / SUBJECT] students. It should let them [WHAT STUDENTS DO] using this content: [PASTE YOUR TERMS / QUESTIONS / DATA DIRECTLY HERE] Keep it to one HTML file, and make it look clean and modern. Ask me 5 questions about the app before you begin coding this project!

Example:

"Build me a single-page web app called Bio Flashcards for my 9th grade biology students. It should let them flip cards to reveal definitions of cell parts, using this content: Mitochondria -> powerhouse of cell, Nucleus -> control center... Keep it to one HTML file."
