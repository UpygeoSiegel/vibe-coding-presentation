05 / The Technology

Meet Claude Code

Your terminal-based software assistant.

1

Local Files: Writes real HTML, CSS, and JS directly into a folder on your computer.

2

Instant Previews: Sets up live, auto-reloading web previews so you can view adjustments as they happen.

3

Self-Healing Code: Monitors and catches logical errors, debugging dynamically inside the files.

terminal -- claude-code

💡 Crucial Difference:

claude.ai chat gives snippets & suggestions.

Claude Code takes action—making local files, hosting local web servers, and maintaining folder structures inside your directory.

08 / Setup Phase 1

Install Claude Code

Launch your terminal application and execute your system's installer command below.

🍎

macOS

  1. Launch Terminal via Spotlight search.
  2. Paste the command block below and hit Enter.
  3. Restart Terminal once execution finishes.
curl -fsSL https://claude.ai/install.sh | bash
🪟

Windows

  1. Search and launch PowerShell from the Start menu.
  2. Paste the snippet below and execute with Enter.
  3. Restart PowerShell once installation completes.
irm https://claude.ai/install.ps1 | iex

10 / Anatomy

Anatomy of an App Prompt

Excellent education applications are engineered with five critical variables in the prompt:

🎯

1. Goal

The precise action students execute in the sandbox environment.

👥

2. Audience

The specific cognitive profile or grade level constraints.

📖

3. Content

The actual terms, equations, vocabulary, or systems matched.

🛑

4. Limits

Architectural limitations: a single, fully-packaged local file.

🎨

5. Aesthetic

The specific styling, tone, and active display visual elements.

11 / Starting Blueprint

The Golden Prompt Template

Copy this blueprint directly into your active Claude Code session to prompt your project:

Build me a single-page web app called [APP NAME] for my [GRADE / SUBJECT] students. It should let them [WHAT STUDENTS DO] using this content: [PASTE YOUR TERMS / QUESTIONS / DATA DIRECTLY HERE] Keep it to one HTML file, and make it look clean and modern. Ask me 5 clarifying questions about the app features before you begin coding!

Example Execution: "Build me a single-page web app called Bio Flashcards for my 9th grade biology students. It should let them flip cards to reveal definitions of cell parts using this content: Mitochondria -> power center, Nucleus -> command central. Keep it to one HTML file."
